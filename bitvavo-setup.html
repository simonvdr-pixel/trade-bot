Vul je API sleutels in om te starten
📋 API key aanmaken op Bitvavo
1
Ga naar bitvavo.com en log in op je account
2
Klik rechtsboven op je naam → Instellingen → API
3
Klik "API sleutel toevoegen" en geef het een naam (bijv. "TradeBot")
4
Vink aan:
✓ Weergeven (verplicht)
✓ Handelen (voor automatische orders)
✗ Opnemen — NOOIT aanzetten!
5
Kopieer de API Key én API Secret — het secret zie je maar één keer!
Ga naar Bitvavo API instellingen →
